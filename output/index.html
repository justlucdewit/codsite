Why COD?

Cod is a concatinative stackbased language
Concatinative
Concativate programming languages are an untraditional pointer free programming paradigm with 'words' that denote functions that are called one by one.
Stack based
COD primarily uses a stack based datastructure, a last in first out data structure. It combines this with a heap & variables to make the life of the programmer easier.
Open source
The source code of the COD compiler and all its tools are opensource and can be read, modified, and redistributed in any way you like using its repository on Github.

about COD
COD is a stack based concatinative programming language, It has an untraditional paradigm yet uses abstraction and other features to try to be as practical as possible. Give it a try, have fun, and leave feedback about your experience
